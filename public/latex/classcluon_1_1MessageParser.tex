\doxysubsection{Detailed Description}
This class provides a parser for message specifications in .odvd format. The format is inspired by Google Protobuf (\href{https://developers.google.com/protocol-buffers/}{\texttt{ https\+://developers.\+google.\+com/protocol-\/buffers/}}) but simplified to enforce backwards and forwards compatibility next to numerical message identifiers.

This message specification format is also used by Open\+Da\+VINCI (\href{http://code.opendavinci.org}{\texttt{ http\+://code.\+opendavinci.\+org}}).

The parser is based on \href{https://github.com/yhirose/cpp-peglib}{\texttt{ https\+://github.\+com/yhirose/cpp-\/peglib}}.
