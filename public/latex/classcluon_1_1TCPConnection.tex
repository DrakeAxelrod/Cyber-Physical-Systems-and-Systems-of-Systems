\doxysubsection{Detailed Description}
To exchange data via TCP, simply include the header {\ttfamily \#include \texorpdfstring{$<$}{<}cluon/\+TCPConnection.\+hpp\texorpdfstring{$>$}{>}}.

Next, create an instance of class {\ttfamily \mbox{\hyperlink{classcluon_1_1TCPConnection}{cluon\+::\+TCPConnection}}} as follows\+: {\ttfamily \mbox{\hyperlink{classcluon_1_1TCPConnection}{cluon\+::\+TCPConnection}} connection(\char`\"{}127.\+0.\+0.\+1\char`\"{}, 1234, new\+Data\+Delegate, connection\+Lost\+Delegate);}. The first parameter is of type {\ttfamily std\+::string} expecting a numerical IPv4 address, the second parameter specifies the TCP port, from which data shall be received from, the third paraemter is of type {\ttfamily std\+::function} that is called whenever new bytes are available to be processed, and the last parameter is of type {\ttfamily std\+::function} that is called when the connection is lost.

The complete signature for the new\+Data\+Delegate function is {\ttfamily std\+::function\texorpdfstring{$<$}{<}void(std\+::string \&\&, std\+::string \&\&, std\+::chrono\+::system\+\_\+clock\+::time\+\_\+point \&\&) noexcept\texorpdfstring{$>$}{>}}\+: The first parameter contains the bytes that have been received, the second parameter containes the human-\/readable representation of the sender (X.\+Y.\+Z.\+W\+:ABCD), and the last parameter is the time stamp when the data has been received.

The complete signature for the connection\+Lost\+Delegate function is {\ttfamily std\+::function\texorpdfstring{$<$}{<}void() noexcept\texorpdfstring{$>$}{>}}.

To finally send data, simply call the method {\ttfamily send} supplying the data to be sent\+: {\ttfamily connection.\+send(std\+::move(\char`\"{}\+Hello World!\char`\"{})}. Please note that the data is supplied using the {\itshape move}-\/semantics. The method {\ttfamily send} returns a {\ttfamily std\+::pair\texorpdfstring{$<$}{<}ssize\+\_\+t, int32\+\_\+t\texorpdfstring{$>$}{>}} where the first element returns the size of the successfully sent bytes and the second element contains the error code in case the transmission of the data failed.
