\doxysubsection{Detailed Description}
This class provides an interface to an Open\+Da\+VINCI v4 session. An Open\+Da\+VINCI v4 session allows the automatic exchange of time-\/stamped Envelopes carrying user-\/defined messages usually using UDP multicast. A running \mbox{\hyperlink{classcluon_1_1OD4Session}{OD4\+Session}} will not receive the bytes that itself has sent to other microservices.

There are two ways to participate in an Open\+Da\+VINCI session. Variant A is simply calling a user-\/supplied lambda whenever a new Envelope is received\+:
