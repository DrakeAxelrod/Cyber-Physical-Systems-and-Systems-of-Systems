\doxysubsection{Detailed Description}
This class provides a visitor to transform a message into CSV with user-\/specified delimiters and optional column headers\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{MyMessage msg;}
\DoxyCodeLine{\textcolor{comment}{// Set some values in msg.}}
\DoxyCodeLine{}
\DoxyCodeLine{\mbox{\hyperlink{classcluon_1_1ToCSVVisitor}{cluon::ToCSVVisitor}} \mbox{\hyperlink{classcluon_1_1ToCSVVisitor_a47ad4c1e54208767ce93f469754f73a3}{csv}}\{\textcolor{charliteral}{','}, \textcolor{keyword}{true}\};}
\DoxyCodeLine{msg.accept(\mbox{\hyperlink{classcluon_1_1ToCSVVisitor_a47ad4c1e54208767ce93f469754f73a3}{csv}});}
\DoxyCodeLine{}
\DoxyCodeLine{std::cout << \mbox{\hyperlink{classcluon_1_1ToCSVVisitor_a47ad4c1e54208767ce93f469754f73a3}{csv}}.csv() << std::endl;}

\end{DoxyCode}


Subsequent use of this visitor will append the data (please keep in mind to not change the visited messages in between as the generated CSV data will be messed up otherwise).
